Alpha‑ea — Documentation (Envato-ready)

This package is prepared for Envato Market submission. It contains a premium, original template inspired by luxury/event websites. Below are step-by-step installation, customization and publishing notes you must follow before submission.

  • index.html — main landing page (hero, gallery, services, contact)

2. How to replace images (step-by-step)

Envato requires all images in items to be licensed. Do NOT upload screenshots or images from Google Image results unless they're properly licensed. We recommend using high-resolution images from Unsplash or Pexels (both free for commercial use). Below are suggested searches and how to replace:

  1. Recommended searches (Unsplash / Pexels):
    luxury yacht, regatta, yacht aerial, luxury event.
    Example sources: Unsplash — luxury yacht. citeturn0image0turn0image3turn0image11
  2. Download full-resolution JPEG or WebP images. Prefer WebP for smaller sizes. Keep hero image width ≥ 2400px for a crisp preview.
  3. Replace files in assets/img/ with the chosen images using the exact filenames used in the template (e.g. hero-large.jpg, gallery1.jpg, etc.).
  4. Credit/attribution: While Unsplash/Pexels do not require attribution, keep an internal credits list in your documentation as best practice.

4. Editing/branding

To change brand name or logo:

  1. Edit the brand text inside index.html (search for Alpha‑ea).
  2. Replace assets/img/favicon.png with your favicon (32×32 or 64×64).
  3. If you replace fonts, ensure to include license info and update the <link> tags in the <head>.

5. Technical checklist (Envato reviewers will check these)

  • All HTML files validate with W3C HTML Validator.
  • No console errors in Chrome/Firefox/Safari when demo is opened.
  • All images used in the preview are licensed and optimized.
  • Documentation is included and explains installation and customization.
